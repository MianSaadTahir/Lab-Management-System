# LMS

A Windows Forms Application developed in C# with SQL database integration. It streamlines the management of student evaluations for a course lab, allowing teachers to efficiently handle various academic components related to Outcome-Based Education (OBE).

## Features
- Add, edit, and delete student records.
- Organize and maintain CLOs associated with courses.
- Create and modify rubrics to define assessment criteria.
- Create and manage assessments linked to CLOs and rubrics.
- Define performance levels within the rubric for nuanced evaluations.
- Record evaluations for students based on their assessment performance.
- Generate crystal PDF reports, including CLO-wise and assessment-wise class results.

## Usage
1. Clone the repository:
   `git clone https://github.com/miansaadtahir/LabManagementSystem.git`
2. Navigate to:
   `cd .\script\`
3. Run the `ProjectBScripts.sql` in SSMS
4. Navigate to the project directory:
   `cd .\LabManagementSystem\ProjectB\bin\Debug`
5. Launch the `ProjectB.exe` file.
